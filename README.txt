Intégrez une carte interactive des offres de Pôle emploi

== Description ==

Ce widget permet de bénéficier d’une interface préconstruite pour requêter et afficher sur une carte les offres d'emploi issues de l’API Offres d’emploi.

Ce widget est interfacé au système d'informations de Pôle emploi, il authentifie l'application qui consomme la base de données des offres d'emploi.

Vous devez disposer d'un compte sur [www.pole-emploi.io](https://francetravail.io) et vous inscrire gratuitement comme utilisateur de l'[API Offres d'emploi](https://francetravail.io/data/api/offres-emploi) pour faire usage de ce widget.

== Installation ==
Installez ce plugin en 4 étapes :

1. Créez un compte sur francetravail.io

2. Créez une application dans votre compte, puis souscrivez à l'API Offres (gratuit) depuis le catalogue des API

3. Paramétrez le widget dans l'interface d'administration de votre site WordPress (Outil / Widget Offres). Vous devez renseigner obligatoirement votre identifiant / clé secrète de l'application crée à l'étape 2. Attention, ne mettez pas ici les identifiants de votre compte francetravail.io

4. Insérez le shortcode [widget-offres-pe]
